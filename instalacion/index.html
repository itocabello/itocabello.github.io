Descripción

En esta sección se describen los pasos necesarios para preparar el entorno de trabajo que se utilizará durante el desarrollo de los ejercicios del curso.

El objetivo es disponer de un entorno homogéneo que permita ejecutar todas las prácticas propuestas utilizando las herramientas seleccionadas para el procesamiento paralelo de datos.
