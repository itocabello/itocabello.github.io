4.2.1 Instalación de Docker y Docker Compose

Como paso previo al despliegue de la infraestructura Big Data, es necesario preparar el entorno de ejecución mediante la instalación de Docker Engine y Docker Compose, herramientas encargadas de la creación, administración y orquestación de los contenedores que forman el clúster Hadoop.

En este proyecto, Docker proporciona el motor de virtualización ligera sobre el que se ejecutan todos los servicios, mientras que Docker Compose permite definir la infraestructura completa mediante un único archivo docker-compose.yml, simplificando el despliegue y la gestión del entorno distribuido.

El fichero docker-compose.yml utilizado durante el desarrollo del proyecto puede descargarse desde el repositorio GitHub del proyecto para reproducir el despliegue completo del clúster Hadoop.

Instalación de Docker Engine

Una vez instalado Ubuntu 22.04, se actualizan los repositorios del sistema y se instalan Docker Engine y los servicios necesarios para su ejecución automática.

Comandos de instalación 
sudo apt update
sudo apt install -y docker.io
sudo systemctl enable docker
sudo systemctl start docker

Verificación de la instalación

Una vez finalizada la instalación, es recomendable comprobar que Docker se ha instalado correctamente verificando la versión disponible en el sistema.

Comandos de comprobación 
docker --version
docker compose version

Resultado

Tras completar este proceso, el sistema queda preparado para descargar las imágenes Docker del proyecto y desplegar automáticamente toda la infraestructura Hadoop mediante Docker Compose.
